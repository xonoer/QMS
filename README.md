## car
针对车载号码牌的一个想法。

同学们，生活当中有没有发现当你停车时候不一定有车位，然后停车挡住了其他车辆的情况发生？

换位思考你的车辆被挡住了呢？我在北京的一个小区里租的房子，车位供给紧张，有时候不得不挨着其他车辆停放，起初我也买了车载号码牌，日久老化掉了。。。

果断搜淘宝有摆放式号码牌，而且还带手机支架（心动半天，差点剁手买了，然而并没有什么卵用。我比较主张提供简约突出功能。）
